# Diagram Exports

PNG files in this directory are exported from the `.excalidraw` source files in the parent directory.

## How to export

1. Install the [Excalidraw VS Code extension](https://marketplace.visualstudio.com/items?itemName=pomdtr.excalidraw-editor)
2. Open a `.excalidraw` file in `docs/diagrams/`
3. Use the export button (or right-click menu) to export as PNG
4. Save the PNG to this directory with the same base name (e.g., `architecture.excalidraw` -> `architecture.png`)

The exported PNGs are referenced by `README.md` in the project root.
